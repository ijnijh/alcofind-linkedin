# ALCOFIND LinkedIn 포스팅 작성앱

개인 계정(주 2회 · 한국어) + 회사 페이지(주 1회 · 영문 전용) LinkedIn 포스팅을
**제목 + 본문 + 브랜드 이미지까지 한 번에 자동 생성**하는 Streamlit 앱.

## 운영 리듬

| 트랙 | 언어 | 빈도 | 슬롯 | 시리즈 |
|---|---|---|---|---|
| **개인** (김진호) | 한 (+영 월1) | 주 2 | 화·목 09:00 | 현장노트 / 산업 인사이트 / Korea Brief / BD 노트 |
| **회사** (ALCOFIND) | **영문 전용** | 주 1 | 수 10:00 KST | Industry Pulse / Company News |

두 트랙 모두 최종 목적은 **국내외 인바운드 리드 확보**.

영문 트랙은 모든 출력이 **한·영 병기**로 나옵니다 (EN = 게시용, KO = 검수용).

## 🚀 처음 실행하기

### 1) Anthropic API 키 발급 (본문 자동 생성용)

본문 생성은 Claude API를 호출하므로 키가 필요합니다. **무료 가입 → 키 발급 → 충전(최소 $5)** 순.

1. **[console.anthropic.com](https://console.anthropic.com)** 가입 (Google/이메일 모두 가능)
2. 좌측 **Plans & Billing** → **Add credits** → 최소 $5 충전 (카드 결제). 한 번 충전한 크레딧은 만료 없음.
3. **Settings → API Keys → Create Key** → 키 이름 자유롭게 → **Generated key 복사** (다시 표시 안 됨)
4. 앱 폴더에 `.env` 파일 생성, 한 줄:

```
ANTHROPIC_API_KEY=sk-ant-api03-여기에붙여넣기
```

5. 앱 실행 시 자동 로드. (`.env.example` 파일 참고)

> 🔒 `.env`는 절대 외부에 공유하지 마세요. 키가 노출되면 즉시 Console에서 revoke 후 재발급.

### 2) 예상 비용

포스트 1건당 (입력 약 2,000 토큰 · 출력 약 1,500 토큰 가정):

| 모델 | 비용/포스트 | 월 12~16건 | 권장 |
|---|---|---|---|
| **Haiku 4.5** | ≈ ₩8 | ₩100~130 | 초안 빠르게 |
| **Sonnet 4.6** (기본) | ≈ ₩40 | ₩500~650 | **균형 — 권장** |
| **Opus 4.7** | ≈ ₩200 | ₩2,400~3,200 | 까다로운 포스트만 |

Sonnet 기본으로 운영하고, 시그니처 시리즈 첫 회나 중요 발표는 Opus로 한 번 더 돌리는 워크플로우가 가장 가성비 좋습니다.

### 3) 앱 실행

탐색기에서 `run.bat` 더블클릭. 또는 PowerShell:

```powershell
& "C:\Users\jhkim\Documents\Claude code\linkedin\app\run.bat"
```

처음 실행은 venv 자동 생성 + 의존성 설치(1~2분), 이후엔 즉시 기동.
브라우저가 `http://localhost:8501`을 자동으로 엽니다.

## 📋 사용 흐름

1. **개인** 또는 **회사** 탭 선택
2. 시리즈 → 주제 → 핵심 인사이트 입력 (선택: 익명 케이스 · 추가 출처 · CTA 직접 지정)
3. 이미지 템플릿 선택 (Hook / Quote / Stat). 후킹 텍스트는 자동 채워짐 — 비워두면 OK
4. **✨ 포스트 자동 생성** 클릭 → 8~20초 후 결과 표시
5. 결과 화면:
   - 📌 **TITLE** 헤드라인 (영문이면 KO 참고도)
   - 📝 **본문 전체** (영문 트랙은 EN 탭 / KO 탭으로 분리)
   - 🖼️ **브랜드 이미지** + PNG 다운로드 버튼
   - 🧠 의도 노트(검수용, 접힘)
6. 본문 복사 → LinkedIn 게시 / 이미지 다운로드 → 첨부

키가 없으면 **🖼️ 이미지만 생성** 버튼으로 이미지만 만들 수 있고, 본문 생성용 프롬프트는 별도로 표시되어 Claude 채팅에 붙여넣어 사용할 수 있습니다.

## ⚙️ 설정 변경

| 변경 항목 | 파일 |
|---|---|
| 시리즈 추가 · 톤 조정 · CTA 기본값 | `config.py` (PERSONAL_SERIES / COMPANY_SERIES) |
| 브랜드 컬러 · 폰트 | `config.py` (BRAND) |
| 모델 라인업 | `config.py` (MODELS) |
| 본문 길이 가이드 · 규칙 | `prompt_builder.py` (SYSTEM_NOTE) |
| 이미지 카드 디자인 | `image_renderer.py` |

앱은 파일 저장 시 Streamlit 자동 리로드. 새로고침만 하면 반영됩니다.

- 앱의 **📜 히스토리** 탭에서 최근 30건 조회·재다운로드 가능

## 🛟 문제 해결

| 증상 | 원인 / 조치 |
|---|---|
| "API 키 없음" 경고 | `.env` 파일 위치 확인 (`app/` 폴더), 또는 사이드바에서 임시 입력 |
| 응답 파싱 실패 | 사이드바에서 모델을 한 단계 높여 재시도 (Haiku → Sonnet). 또는 "원본 LLM 응답" 디버그 펼침 확인 |
| 비용 한도 초과 | console.anthropic.com → Plans & Billing 에서 한도 확인 후 추가 충전 |
| 브라우저가 안 열림 | `http://localhost:8501` 직접 입력 |
| Streamlit 종료 | 터미널 창에서 `Ctrl+C` |
